## ❓ Other ways to help

If you can't sponsor financially, these all help just as much:

- ⭐ Star the repo
- 🐛 Report bugs with reproduction steps
- 💡 Open feature-request issues
- 📢 Tell other Obsidian users about it
- 📝 Improve the README, write a tutorial, translate it

如果不方便赞助，这些方式同样有帮助：

- ⭐ 给仓库点 Star
- 🐛 提 issue 报告 bug（带复现步骤更佳）
- 💡 提 feature request
- 📢 在朋友圈 / 即刻 / B 站介绍给其他 Obsidian 用户
- 📝 改进 README、写教程、翻译
